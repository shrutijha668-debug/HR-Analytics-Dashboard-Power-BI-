# HR Analytics Power-BI Dashboard

## Project Overview

In this project, a Power BI dashboard was designed to analyze employee data, focusing on workforce trends, employee attrition, and organizational performance. The dashboard provides valuable insights into key areas such as employee count, attrition rate, salary distribution, and demographic analysis.

## Objectives

Build an interactive Power BI dashboard to visualize key HR data and metrics.
Display workforce performance across various dimensions, including attrition, demographics, and salary distribution.
Offer actionable insights to support better decision-making in human resource management.

## Dataset

The data for the dashboard was sourced from employee datasets containing information about age, gender, department, salary, and attrition status.

## Features of the Dashboard

Workforce Overview: Visualizations showing total employees, active employees, attrition count, and attrition rate.
Attrition Analysis: Insights into employee attrition by department and gender.
Demographic Insights: Analysis of employee distribution based on age groups and gender.
Salary Analysis: Visual representation of employees across different salary slabs and categories.
